# Automated Historical Content Publisher

## Overview
An AI-powered n8n workflow that automatically discovers historical events that occurred on the current date, generates engaging LinkedIn content using Google Gemini AI, and publishes the content to LinkedIn.

## Workflow Diagram

```text
Schedule Trigger
      │
      ▼
Google Gemini AI
(Find Historical Fact)
      │
      ▼
Generate LinkedIn Post
      │
      ▼
LinkedIn Publisher
      │
      ▼
Published Post
```

## Technologies Used

- n8n
- Google Gemini AI
- LinkedIn API

## Features

- Automated daily execution
- Historical fact discovery
- AI-generated content
- LinkedIn auto publishing

## Use Case

Helps content creators, educators, and history enthusiasts automatically share historical events and facts on social media.
